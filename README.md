# GeographyTeachingPlugin
地理教学用-神秘小插件合集

## 地转偏向力与大气环流演变模型 (大气环流.html)

这是一个纯前端（HTML+原生JS）的交互式地理教学演示组件，无需联网即可运行。

**功能特点：**
- **滑动交互**：拖动滑块实时模拟地球自转（地转偏向力）从无到有的过程。
- **动态推演**：直观展示气流如何在30°纬度堆积下沉，将“单圈环流”切断为“三圈环流”。
- **双重视角**：左侧/中心展示球面风向偏转轨迹，右侧展示大气对流层垂直剖面流向。

**使用方法：**
1. 下载 `大气环流.html` 文件。
2. 双击在浏览器中全屏打开。
3. 或在 PowerPoint 中通过“插入 -> 对象 -> 由文件创建”嵌入使用。
